============
INTRODUCTION
============


Stratio Explorer is an Interactive Web interpreter  to Apache Crossdata, Stratio Ingestion,
Stratio Decision,Markdown, Apache Spark, Apache Spark-SQL and command Shell.

Stratio Explorer is based in Apache Zeppelin .

With Stratio Explorer you can make paragraphs that connect with diferents platforms ,
write insructions to the platform and visualize results in your web browser with 
formats (text and diferents type graphics)
